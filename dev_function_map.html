核心业务逻辑关系总览

主流程依赖链
标准发布 → 举证申请 → 匹配度更新 → 晋升达标 → 晋升申请 → 评审 → 预算 → 生效
互斥关系汇总
  • 升级 ⊕ 升等：同一申请二选一
  • 同子类 ⊕ 跨子类 ⊕ 绿通：通道互斥
  • 预算通过 ⊕ 排队等待：结果互斥
关键计算规则
  • 匹配度 = Σ(举证项 × 权重)
  • 升级：匹配度 ≥ 100%
  • 升等：匹配度 ≥ 等级阈值 (20%/40%/60%/80%)
